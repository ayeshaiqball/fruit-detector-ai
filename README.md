# 🍌🍍🐉 Fruit Classifier AI (Streamlit + ML + DL + Roboflow)

An AI-powered web application that classifies fruits into **three categories: Banana, Pineapple, and Dragon/Snake Fruit** using Machine Learning, Deep Learning, and a Roboflow-trained model.

---

## 🚀 Project Overview

This project is a Streamlit-based interactive web application that predicts the type of fruit from an input image.

It combines multiple AI approaches:
- Machine Learning
- Deep Learning (CNN)
- Roboflow Pre-trained Model
- Agentic AI logic for decision handling

---

## 🍎 Classes Supported

The model can classify the following fruits:

- 🍌 Banana  
- 🍍 Pineapple  
- 🐉 Dragon/Snake Fruit  

---

## 🧠 AI Models Used

### 1. Machine Learning Model
- Used for basic feature-based classification

### 2. Deep Learning Model (CNN)
- Used for image-based recognition

### 3. Roboflow Model
- Pre-trained object detection model
- Improves prediction accuracy

### 4. Agentic AI Layer
- Handles prediction flow and decision logic

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/fruit-classifier-ai.git
```

2. Go to project folder:

```bash
cd fruit-classifier-ai
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run Streamlit app:

```bash
streamlit run app.py
```
